Social Spark Suggestion App

📱 Overview
Early mornings might mean a quick message to someone you know. Built with Kotlin inside Android Studio, this tool runs quietly in the background. Instead of waiting around, it nudges you when timing feels right. Time entries shape what kind of gesture shows up next. Sometimes that means calling an old friend. Other moments suggest just sending a short note. Each idea fits loosely into parts of your daily rhythm. Social steps come one at a time, shaped by how you log hours. Quiet prompts appear only after you feed the system bits of schedule.

🎯 Purpose
Busy schedules often pull people apart. Yet tiny moments can keep bonds alive. This tool offers small prompts that spark real conversations. Instead of scrolling endlessly, someone might share a memory. A question appears. It feels light, not forced. Connection happens without pressure. Ideas show up fast, like thoughts between tasks. Not everything needs time. Sometimes just one word leads somewhere warm.

Features
A person types what part of day it is - like morning or afternoon. That entry goes into a box meant just for that info. Sometimes they might write dinner instead. Each choice fits how folks talk about time normally. The system takes whatever gets put there
- Suggestion display based on input
- Use of if statements for decision making
- Reset button to clear input and output
- Error handling for invalid input

How It Works
When someone types a time, the system checks it right away. If the value fits one range, another path runs instead. Each category has its own response ready. A fitting idea shows up after sorting happens behind the scenes.

Should someone type something wrong, the system responds by asking for a proper time entry. Not quite right? It nudges toward accuracy without delay. Wrong input shows a note - fix it, try again. A mistyped moment brings up guidance instantly. Entry fails? The message appears: adjust and reenter.
